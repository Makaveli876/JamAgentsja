# Yard Wire | Premium WhatsApp Status Assets

Yard Wire is a high-fidelity Progressive Web App (PWA) designed for the Jamaican market to generate premium marketing assets for WhatsApp Status.

## Tech Stack
- **Frontend**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS 4
- **AI**: Google Gemini 1.5 Flash (via `@google/genai`)
- **Icons**: Lucide React

## Key Features
- **Cyber-Island Aesthetic**: High-fidelity glassmorphism and neon accents.
- **AI Optimization**: One-click listing refinement powered by Gemini.
- **Instant Preview**: Real-time asset rendering with client-side image preview.
- **PWA Ready**: Optimized for mobile standalone experience.

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Environment Setup**:
   Create a `.env.local` file and add your Gemini API key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

3. **Run Development Server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000)

## Design Guidelines
- **Primary Color**: Yard Cyan (#00f2ff)
- **Secondary Color**: Yard Purple (#bc13fe)
- **Accent Colors**: Yard Gold (#ffd700), Yard Green (#00ff41)
- **Aesthetic**: Dark mode, heavy blurs, subtle neon halos.
