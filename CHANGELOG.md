# Changelog

All notable changes to the Jam Agents project will be documented in this file.

## [v0.3.0] - 2026-01-11
### Added
- **Image Upload Logic**: Instant client-side preview using `URL.createObjectURL`.
- **JamRock Layout**: Complete refactor of `AssetPreview` to a high-fidelity vertical marketing format.
- **Documentation**: Added `README.md` and `CHANGELOG.md`.

## [v0.2.0] - 2026-01-11
### Added
- **AI Integration**: Integrated Google Gemini 1.5 Flash for listing optimization.
- **Aesthetic v2**: Added dusky landscape background and gold-to-green gradient headers.
- **Hydration Fix**: Resolved React hydration mismatches in the main studio interface.

## [v0.1.0] - 2026-01-11
### Added
- **Initial Scaffold**: Next.js 14 project setup with Tailwind 4.
- **Design System**: Core "Cyber-Island" glassmorphism and color tokens.
- **Studio Interface**: Basic split-screen Control/Preview layout.
